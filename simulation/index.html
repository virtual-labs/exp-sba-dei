📚 Service Management Guide

  1. Click Create New Service, enter details for Service1, and start it.
  2. Verify Service1 is running by checking the logs for active status.
  3. Click Create New Service again, configure Service2 with a different IP and port, and start it.
  4. Check logs to confirm communication between Service1 and Service2.
  5. Open Service2 Terminal and run ping <Service1_IP>.
  6. Verify connectivity by ensuring packets are sent and received with 0% loss.
